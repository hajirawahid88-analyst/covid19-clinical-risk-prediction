## Overview

This project presents a machine learning pipeline for predicting clinical outcomes of COVID-19 patients using real-world healthcare data. The work was completed as part of my Master of Science in Computer Science (MSCS) at the National University of Sciences and Technology (NUST), Pakistan.

The project focuses on developing accurate and interpretable machine learning models that can support healthcare professionals in early risk assessment and clinical decision-making.

---

## Objectives

- Develop machine learning models for COVID-19 clinical risk prediction.
- Preprocess and integrate heterogeneous clinical datasets.
- Compare the performance of different machine learning algorithms.
- Improve model interpretability using Explainable AI (XAI).
- Support evidence-based clinical decision-making.

---

## Dataset

The research was conducted using anonymized clinical data collected in collaboration with Rawalpindi Medical University.

**Data sources included:**
- Holy Family Hospital
- Azad Jammu and Kashmir Health Department

The original dataset contains approximately 6,000 patient records.

**Note:** The original dataset is confidential and is **not included** in this repository. A sample dataset is provided only for demonstration purposes.
